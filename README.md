# Haberler 📰

Türkçe bir haber uygulaması. SwiftUI ile geliştirilen, iOS için bir haber okuma deneyimi.

## Hakkında

Haberler; güncel haberleri listeleyen, kategorilere ayıran ve detaylarını okumanı sağlayan bir iOS uygulaması. Proje henüz geliştirme aşamasında.

## Özellikler

- [ ] Haber listesi ekranı
- [ ] Haber detay sayfası
- [ ] Kategoriler
- [ ] API'den canlı haber çekme
- [ ] Arama
- [ ] Kaydedilenler / favoriler

## Teknolojiler

- **Swift** & **SwiftUI**
- **Xcode** (iOS)

## Kurulum

```bash
git clone https://github.com/MERVEAKCAKAYA/Haberler.git
cd Haberler
open Haberler.xcodeproj
```

Ardından Xcode üzerinden bir simülatör seçip **Run** (`⌘R`) ile çalıştırabilirsin.

## Gereksinimler

- Xcode 15+
- iOS 17+

## Lisans

Bu proje kişisel bir çalışmadır.
